Brändi Dog 2.0 Lobby Manual

Das Hauptfenster

In der Chat-Lobby kannst du dich mit anderen Clients unterhalten. Im Server-Tab (links, oben) siehst du, welche Clients sich wann ein- bzw. ausgeloggt haben; ausserdem wird hier auch der gemeinsamer Chat angezeigt. Im Textfeld darunter (links, unten) kannst du dein eigenen Nachrichten eintippen und durch drücken der Enter-Taste abschicken.

Im Users-Tab (rechts) werden die Clients angezeigt, die zur Zeit online im Server sind. Klicke einen User an, um ein privates Chatfenster mit ihm zu öffnen. Es wird neben dem Server-Tab geöffnet.

Im Lobbies-Tab (rechts) werden die aktuellen Game-Lobbies angezeigt. Klicke eine Lobby an, um ihr beizutreten. Das Lobby-Chat öffnet sich dann neben dem Server-Tab.


Die Menüleiste

Es gibt vier Menüs in der Leiste am oberen Rand:

  • Menü Client
    • Connect: Hiermit kannst du dich mit einem anderen Server verbinden. Die momentane Verbindung wird geschlossen.
    • Change Nickname: Hiermit kannst du deinen Namen im Chat ändern.
    • Close: Hiermit wird die Verbindung zum Server getrennt und das Programm geschlossen.
  • Menü Game
    • Fullscreen: Schaltest du diesen Modus ein, werden Spiele im Vollbildmodus gestartet.
  • Menü Server
    • Scoreboard: Hiermit zeigst du das Scoreboard mit den aktuellen Highscores auf dem Server an.
    • Info: Hiermit zeigst du die IP-Adresse und den Port des Servers an.
    • Create Lobby: Hiermit erstellst du eine Lobby und trittst ihr bei. So kannst du ein Spiel starten.
  • Menü Help
    • Lobby Usage: Öffnet dieses Fenster.
